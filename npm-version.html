npm version

Published: Thu 30 August 2018
By yinxianwei

In 随笔.

版本示例：1.2.3 MAJOR.MINOR.PATCH:

1.2.3
┬ ┬ ┬
│ │ │
│ │ └ 补丁版本：bug修复，向后兼容
│ └── 次要版本：新增功能，向后兼容
└──── 主要版本：修改不向后兼容的api
示例
  1.2.3
 =1.2.3
 >1.2.3
 <1.2.3
>=1.2.3

Note：1.2.3-rc1不在范围内
版本范围
声明 结果 备注
~1.2.3 is >=1.2.3 <1.3.0
^1.2.3 is >=1.2.3 <2.0.0
^0.2.3 is >=0.2.3 <0.3.0 (0.x.x 不限定后两位版本)
^0.0.1 is =0.0.1 (0.0.x不限定第三位版本)
^1.2 is >=1.2.0 <2.0.0 (等同于\^1.2.0)
~1.2 is >=1.2.0 <1.3.0 (等同于~1.2.0)
^1 is >=1.0.0 <2.0.0
~1 same
1.x same
1.* same
1 same
* 最新版本
x same
版本范围
1.2.3 - 2.3.0   为 >=1.2.3 <=2.3.4
1.2.3 - 2.3     为 >=1.2.3 <2.4.0
1.2.3 - 2       为 >=1.2.3 <3.0.0
1.2 - 2.3.0     为 1.2.0 - 2.3.0

横线右边 2.3 = 2.3.x
横线左边 1.2 = 1.2.0
条件版本范围
>=0.14 <16  并且（空格分开）
0.14.x || 15.x.x    或者（||分开）
Pre-releases

1.2.3-prerelease+build

符号说明
^   版本兼容
~   版本接近
0.x.x   最初版本
1.x.x   公开的版本
