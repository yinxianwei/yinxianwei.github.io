toFixed() 方法使用定点表示法来格式化一个数。

js console.log((0.005).toFixed(2)) -> 0.01 console.log((0.015).toFixed(2)) -> 0.01 console.log((0.025).toFixed(2)) -> 0.03 console.log((0.035).toFixed(2)) -> 0.04 console.log((0.045).toFixed(2)) -> 0.04 console.log((0.055).toFixed(2)) -> 0.06 console.log((0.065).toFixed(2)) -> 0.07 console.log((0.075).toFixed(2)) -> 0.07 console.log((0.085).toFixed(2)) -> 0.09 console.log((0.095).toFixed(2)) -> 0.10
