##### 版本范围

```

1.2.3 - 2.3.0   为 >=1.2.3 <=2.3.4
1.2.3 - 2.3	    为 >=1.2.3 <2.4.0
1.2.3 - 2       为 >=1.2.3 <3.0.0
1.2 - 2.3.0	    为 1.2.0 - 2.3.0

横线右边 2.3 = 2.3.x
横线左边 1.2 = 1.2.0

```
