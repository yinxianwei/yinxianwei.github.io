2026-06-23 AI资讯简报

【摘要】

今日AI领域焦点集中在多代理系统与开源高效方案上，Sakana AI发布Fugu Ultra，通过单一API实现前沿多代理编排，性能匹敌Fable/Mythos且规避出口管制风险。NVIDIA与Micron分别推出液冷节水技术和与Anthropic的战略合作及投资，凸显AI基础设施能效与供应链优化趋势。Claude Code结合GLM-5.2等开源模型降低成本的实践广泛传播，AI Workers与自主循环等代理工作流成为热门讨论，显示行业从单纯模型竞赛转向实际部署效率与生态构建。

【热门事件 Top 10】

  1. Sakana AI 发布 Fugu Ultra 多代理系统

    • 内容：推出单一API的多代理编排系统Fugu Ultra，性能匹配Fable和Mythos，无出口管制风险，支持立即试用。
    • 来源：@SakanaAILabs
    • 链接：https://x.com/SakanaAILabs/status/2068861630327443966

  2. NVIDIA 发布 AI 数据中心液冷节水方案

    • 内容：液冷技术将AI工厂冷却用水从260万加仑/MW/年降至接近零，并支持热能再利用，成为电网资产。
    • 来源：@nvidia
    • 链接：https://x.com/nvidia/status/2069147938098483586

  3. Micron 与 Anthropic 达成战略合作

    • 内容：双方将在内存存储AI架构、Claude企业采用及Series H融资轮展开合作，Micron进行战略投资。
    • 来源：@MicronTech
    • 链接：https://x.com/MicronTech/status/2069046122296070568

  4. 开源 Claude 栈替代每月 200 美元 Opus 支出

    • 内容：通过GLM-5.2、本地模型和Claude Code路由，可将80%任务转移至低成本方案，附免费 playbook 指南。
    • 来源：@AlfieJCarter
    • 链接：https://x.com/AlfieJCarter/status/2069128452976832860

  5. Delos 推出 AI Workers 实现百万 ARR

    • 内容：发布无限AI员工系统，可跨组织执行知识工作，短短几天内达成100万美元年经常性收入。
    • 来源：@pierre_dlgr
    • 链接：https://x.com/pierre_dlgr/status/2069088573563822538

  6. Claude Code 官方免费课程上线

    • 内容：2小时免费课程覆盖CLAUDE.md、Plan Mode、Agent Teams、Skills、MCP等实用内容，含字幕。
    • 来源：@midudev
    • 链接：https://x.com/midudev/status/2069066256355868887

  7. 软件工程师与 Claude 共事的反思诗

    • 内容：一位工程师分享从传统编程到主要与Claude协作、依赖代理管理代码的行业转变感悟。
    • 来源：@CathPoaster
    • 链接：https://x.com/CathPoaster/status/2068925403075420175

  8. Claude Code 新技能实现视频创意拆解

    • 内容：构建技能让Claude Code结合Gemini API分析任意视频，输出钩子、受众、逐帧拆解，成本低廉。
    • 来源：@mikefutia
    • 链接：https://x.com/mikefutia/status/2069213352484880752

  9. Babbily 集成 Sakana Fugu、GLM 5.2 等新模型

    • 内容：平台新增Sakana Fugu Ultra（多代理协调）、Kimi K2.7编码模型和GLM 5.2长上下文工程模型。
    • 来源：@heybabbily
    • 链接：https://x.com/heybabbily/status/2069096824690942200

  10. Fable 5 与 Claude Code 循环工作流讨论

    • 内容：Claude Code 创建者称Fable 5已完成100%编码工作，演示如何通过循环、动态工作流释放模型潜力。
    • 来源：@0xCodez
    • 链接：https://x.com/0xCodez/status/2069086695153160287
