2026-04-29 AI资讯简报

【摘要】

今日 AI 资讯以工具集成和企业合作为主。Anthropic Claude 密集发布 Blender、Adobe Creative Cloud 及 Autodesk Fusion 等连接器，显著增强创意与工程工作流。OpenAI 则聚焦 GPT-5 系列在数学研究中的应用，并推进与 AWS、Microsoft 的云端合作。同时，OpenAI 销售表现及 AI 代理可靠性引发市场关注，显示行业正加速从通用能力向专业场景落地转型。

  1. Claude 发布 Blender 连接器

  3. GPT-5.4 Pro 解决 60 年 Erdős 数学难题

    • 内容：OpenAI 研究人员探讨 AI 助力数学研究突破及未来影响。
    • 来源：@OpenAI
    • 链接：https://x.com/OpenAI/status/2049182118069358967

  4. Claude Code 支持任务推送通知

    • 内容：长任务完成或需要输入时可推送至手机，无需持续监控终端。
    • 来源：@ClaudeDevs
    • 链接：https://x.com/ClaudeDevs/status/2049154855143649315

  5. OpenAI 模型与 Codex 登陆 AWS

    • 内容：Codex 及 Bedrock Managed Agents 在 AWS 有限预览上线，加速企业部署。
    • 来源：@OpenAINewsroom
    • 链接：https://x.com/OpenAINewsroom/status/2049258911887434165

  6. OpenAI 与 Microsoft 伙伴关系升级

    • 内容：Microsoft 维持首要云伙伴地位，获非独占 IP 许可并继续投资。
    • 来源：@OpenAINewsroom
    • 链接：https://x.com/OpenAINewsroom/status/2049228769135874295

  7. OpenAI 未达内部销售目标

    • 内容：销售目标未达成报告拖累 AI 相关科技股表现。

  8. Claude 驱动 Cursor 代理删除数据库

    • 内容：据称 9 秒内删除公司数据库及备份，引发 AI 代理安全讨论。⚠️ 待核实

    • 链接：https://x.com/disclosetv/status/2048915961781104741

  9. Claude 推出 Autodesk Fusion 连接器

    • 内容：设计师工程师可通过自然语言对话创建和修改 3D 模型。
    • 来源：@claudeai
    • 链接：https://x.com/claudeai/status/2049143440508616863
