安卓 position: absolute; position: fixed;和键盘冲突的问题

Published: Thu 27 December 2018
By yinxianwei

In 随笔.

解决方式一

修改为relative或static

解决方式二

设置absolute时固定父元素高度（px）

解决方式三

弹出键盘时隐藏内容

@media (max-height: 500px) {
    .class {
        display: none;
    }
}
