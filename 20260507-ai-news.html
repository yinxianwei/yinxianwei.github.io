2026-05-07 AI资讯简报

【摘要】

今日AI领域基础设施合作密集落地，OpenAI联合AMD、NVIDIA等发布MRC开放网络协议，大幅降低大型训练集群GPU浪费；xAI将Colossus 1超算租借给Anthropic，用于提升Claude推理能力，同时探讨轨道计算可能性。Anthropic举办“Code with Claude”活动，提及无限上下文窗口“coming soon”，引发广泛讨论。具身智能与NeuroAI研究同步推进，Genesis AI推出 dexterous manipulation基础模型，Meta发布NeuralBench基准框架。整体趋势聚焦计算效率优化、代理系统创新与跨公司基础设施共享。

【热门事件 Top 10】

  1. OpenAI 发布 MRC 开放网络协议

    • 内容：与AMD、Broadcom、Intel、Microsoft、NVIDIA合作推出MRC协议，提升AI训练集群可靠性和速度，已部署于自家超算。
    • 来源：@OpenAI
    • 链接：https://x.com/OpenAI/status/2052025532485902368

  2. xAI Colossus 1 租借给 Anthropic

    • 内容：Elon Musk确认将Colossus 1（22万+ NVIDIA GPUs）提供给Anthropic提升Claude，SpaceXAI已迁移至Colossus 2。
    • 来源：@elonmusk
    • 链接：https://x.com/elonmusk/status/2052069691372478511

  3. Claude 无限上下文窗口即将推出

    • 内容：Claude相关事件中提及“infinite context windows coming soon”，被认为实现重大突破，引发热烈讨论。
    • 来源：@marmaduke091
    • 链接：https://x.com/marmaduke091/status/2052060665120977047

  4. xAI API 上线图像生成质量模式

    • 内容：Image Generation Quality Mode现已可用，已为Grok生成超3亿张图像，提升真实感、文字渲染和创意控制。
    • 来源：@xai
    • 链接：https://x.com/xai/status/2052193877675983031

  5. Genesis AI 发布 GENE-26.5 机器人模型

    • 内容：新具身AI公司推出单模型机器人基础模型，支持打蛋、拼魔方、实验室操作等灵巧任务，数据采集高效。
    • 来源：@TheHumanoidHub
    • 链接：https://x.com/TheHumanoidHub/status/2052117692451278893

  6. Meta AI 发布 NeuralBench 框架

    • 内容：推出开源基准测试框架，用于NeuroAI模型评估，支持36个EEG任务和94个数据集，MEG/fMRI即将就绪。
    • 来源：@JeanRemiKing
    • 链接：https://x.com/JeanRemiKing/status/2052034314120896582

  7. NVIDIA 与 Corning 合作扩建 AI 光纤

    • 内容：双方达成多年合作，Corning计划10倍提升美国光学连接产能、增加50%纤维生产并新建3座工厂。
    • 来源：@wallstengine
    • 链接：https://x.com/wallstengine/status/2051988425893630386

  8. DeepMind 与 EVE Online 展开研究合作

    • 内容：Demis Hassabis宣布与Eve Online开发团队合作，将复杂游戏作为AI研究与测试平台。
    • 来源：@demishassabis
    • 链接：https://x.com/demishassabis/status/2052147236952477923

  9. Anthropic 举办 Code with Claude 活动

    • 内容：直播介绍Claude Code新功能，包括多代理编排、advisor策略和托管代理，支持大规模生产部署。
    • 来源：@ClaudeDevs
    • 链接：https://x.com/ClaudeDevs/status/2052055459272761661

  10. OpenAI 安全部署委员会结构争议

    • 内容：听证会证词显示Deployment Safety Board由Microsoft与OpenAI成员主导，Sam Altman具有关键影响力，结构偏向商业部署。
