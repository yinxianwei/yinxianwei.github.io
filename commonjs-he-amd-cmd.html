{жЁЎеқ—еј•з”Ё(require)} {жЁЎеқ—е®ҡд№ү(exports)} {жЁЎеқ—ж ҮиҜҶ(module)}

NodeJSйҒөеҫӘдәҶCommonJSи§„иҢғпјҢеҶҷжі•еҰӮдёӢпјҡ

// foo.js

module.exports = function(x) {
    console.log(x);
};

// index.js

let foo = require('./foo')
foo(1);

CommonJSдё»иҰҒдёәдәҶJSеңЁеҗҺз«ҜеҲ¶е®ҡзҡ„и§„иҢғпјҢдҪҶе№¶дёҚйҖӮз”ЁдёҺеүҚз«ҜпјҢеӣ дёәд»Јз ҒеңЁжҳҜйҖҡиҝҮзҪ‘з»ңеҠ иҪҪпјҢжүҖд»ҘAMDпјҲејӮжӯҘжЁЎеқ—е®ҡд№үпјүеҮәзҺ°пјҡ

define(['dep1','dep2'],function(dep1,dep2){...});

RequireJSе®һзҺ°дәҶAMDи§„иҢғпјҢеҶҷжі•еҰӮдёӢпјҡ

// foo.js

define(function() {
    return function(x) {
        console.log(x);
    };
});

// index.js

define(['foo'], function(foo) {
    foo(2);
});
